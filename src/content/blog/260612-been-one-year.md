I really did create this blog and then proceed to not do anything with it for almost a year straight. Glad that I still have the repo saved, so it is easy enough to pick up again.

I was going around looking for some colors to decide on, and settled on these: the space background from [Nippon Colors](https://nipponcolors.com/#sumi) and the main dark mode palette from [Wada Sanzo Colors](https://www.wada-sanzo-colors.com/). Upon choosing the color for light mode, it reminds me a lot of the Moondrop Space Travel earbuds I own, so this site is hereby named kitan's space travel! pretty simple.

![Moondrop Space Travel earbuds](/images/blog/260615-moondrop-space-travel.jpg)
*Moondrop Space Travel earbuds, which are so cheap and nice that I own two pairs*

I'll be clear that this site's design is done with the help of a LLM Model, so no made by human tag for me. I would like to write more about my opinions on it, but generally, my view on it has been: pretty useful, only for one-off tools, pretty great for suggesting research keywords you can jump off from, but really bad for specific niche searches; prone to make assumptions and run with it without me knowing, but really nice to have it question me to check my own assumptions; very easy to be dependent on, which I already am to an extent and would really like to mitigate.

For the writing, aside from the personal updates, I would like to write about the  research I have done so far on programming and writing computer programs in an effort to resolve the dissasisfaction with the instructions I got from university. I would also like to write a certain programming guide, that is not technical but more on the "how do I do what I aim to do" rules of thumb. I'm not an authority by any means, and my technical prowess is objectively middling, but I hope to write the handholding I wish I had when I started out.

While I don't want to commit a hard schedule for writing lest I fail, I hope making writing easier (making a new post and pushing it here is two computer commands total) would make the process more enjoyable. Also I *really* like the way it looks now, so this site needs to be, and shall be, filled with lots of words.

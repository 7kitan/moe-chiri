## personal site
note: build site with npm run build and deploy to cloudflare with npx wrangler deploy. if forgot how to deploy site, check out [this tutorial](https://ericmaster.ninja/posts/deploy-your-statically-generated-site-to-a-cloudflare-worker/) to deploy to CF workers. (Use workers instead of Pages as CF is deprecating Pages)
## requirements:
- logged into Cloudflare with `npx wrangler login`
